# BlockGame

![Screenshot](https://www.github.com/robertberry/BlockGame/raw/master/res/screenshot.png)

Java/Slick clone of a popular Russian game!

## Controls

Use arrow keys for movement. Space bar performs soft drop, return key hard
drop. P pauses game.
